Summary: Lattitude vs. X


The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis we first pulled data from OpenWetherMap API to assemble a data set of over 500 cities.

After assembling the dataset, we used matplotlib to plot various aspects of weather vs. lattitude. Factors e looked at included temperature, cloudiness, wind speed, and humidity. This site provides the source data and visualizations created as part of the analysis, as well as explanations and descriptions of any trends and correlations witnessed.
